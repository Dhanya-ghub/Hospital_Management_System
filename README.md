# Hospital Management System 🏥

A **Java-based mini project** for managing patients, doctors, and appointments in a hospital.  
The project is built using **Swing UI** for a better visual experience and intuitive interaction.
---
## Features
- Add new patients with details
- View lists of patients, doctors, and appointments
- Pre-book appointments with doctors (ensures no double booking on the same date)
- Supports **12 types of doctors**
- Simple and user-friendly Swing GUI
---
## How It Works
1. **Add Patient:** Add patient details to the system.  
2. **View Lists:** Check patient list, doctor list, and appointment list.  
3. **Book Appointment:** Appointments are pre-booked; the same date cannot be used by another person until the previous booking is deleted.  
---
## Technologies Used
- Java
- Swing (GUI)
- JDBC (Database connectivity)
